# CTI-dataset

An open Chinese Cyber Threat Intelligence (CTI) dataset designed for **entity extraction, relation extraction, and event extraction** research.

## 📌 Dataset Description
The dataset will be published on [Zenodo](https://zenodo.org/) and will be freely available for research purposes.

## ⚙️ Training Models
This project refers to the following open-source models during construction and experiments:
- **Event Extraction**: [DuEE](https://github.com/jxlin98/DuEE)  
- **Entity & Relation Extraction**: [DeepKE](https://github.com/zjunlp/DeepKE)  

## 📝 Annotation Tool
Data annotation is conducted using [Doccano](https://github.com/doccano/doccano).
